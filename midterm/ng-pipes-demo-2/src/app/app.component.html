1. Date Pipes (40 points)

Format the currentDate variable (2024-01-26T15:30:45) using the date pipe.

{{ currentDate | date:'shortDate' }}
{{ currentDate | date:'fullDate' }}
{{ currentDate | date:'MM/dd/yyyy' }}
{{ currentDate | date:'shortTime' }}
{{ currentDate | date:'d' }}
{{ currentDate | date:'w' }}

2. Currency Pipes (24 points)

Format the price variable (42.42424242) using the currency pipe.

{{ price | currency:'USD':'symbol':'1.2-2' }}
{{ price | currency:'USD':'code':'1.2-2' }}
{{ price | currency:'INR':'symbol':'1.2-2' }}
{{ price | currency:'GBP':'symbol':'1.2-2' }}

3. Async Pipes (25 points)

FHandle asynchronous data streams using the async pipe.

{{ arrayStream$ | async }}
{{ transformedData$ | async }}

4. JSON Pipes (30 points)

Use the json pipe to format the provided data structures.

{{ nestedArrays | json }}
{{ complexData | json }}
{{ mixedTypes | json }}

5. Slice Pipes (34 points)

Use the slice pipe to extract specific portions of arrays or strings.

{{ numbers | slice:3:6 }}
{{ numbers | slice:-3 }}
{{ longString | slice:-2 }}
{{ longString | slice:0:5 }}
